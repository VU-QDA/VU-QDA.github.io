COCOON

This project, which is funded by HealthHolland, concerns the prediction of pre-term birth using a variety of data sources (including Electrohysterography) using machine learning.

The project is a collaboration with the Amsterdam UMC (Anna Rietveld, Petra Bakker), Bloomlife Inc., and the Maastricht UMC+ (Pim Teunissen)
